# SCT_CS_1
# 🔐 Caesar Cipher Tool (GUI Based)

## 📌 Overview
This project is a GUI-based Caesar Cipher encryption tool developed as part of a Cyber Security Internship.

It allows users to encrypt, decrypt, and analyze text using classical cryptographic techniques.

---

## 🚀 Features
- 🔑 Encrypt text using Caesar Cipher
- 🔓 Decrypt encrypted text
- 🧠 Brute-force attack simulation (all possible shifts)
- 🖥️ User-friendly GUI using Tkinter
- 📄 Supports alphabets, numbers, and symbols

---

## ⚠️ Security Insight
Caesar Cipher is a weak encryption technique and can be easily broken using brute-force attacks.  
This project demonstrates why modern encryption methods are necessary.

---

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
